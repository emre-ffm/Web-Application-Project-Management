Inventarsystem

Was kann ich alles auf dieser Website durchführen?

Diese Seite soll ihnen die Möglichkeit anbieten,dass Sie sicher und mit wenig aufwand, die Projektarbeiten und Abschlussarbeiten aufnehmen und abrufen können.

Sie können auch die Projekte umändern oder löschen

Sie haben die Möglichkeit sich zu registrieren und mit Ihrem Daten anzumelden.

Sie können die eingetragenen Projekte und Abschlussarbeiten per Suchfunktion suchen.

Ihnen wird eine Statistik mit den vorhandenen Projekten und namen angezeigt.

Sie können Datein in einen Lokalen, vorbestimmten Ordner hochladen

Der User kann auch ein Foto von sich, in ein Ordner oder in eine Datenbank hochladen,falls der User kein eigenes Foto hat,so kann der User ein Foto von sich auf der Webseite aufnehmen und herunterladen.

Mit User rechten haben Sie leider kein Zugriff auf alle Funktionen.
